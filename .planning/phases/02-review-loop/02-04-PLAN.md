Props interface:
```typescript
interface ReviewItem {
  itemId: string;
  title: string;
  content: string;
  domain: string;
  source: string | null;
  tags: string[];
  reviewStateId: string;
  nextReviewAt: string;
  stability: number;
  difficulty: number;
  reviewCount: number;
}

interface ReviewCardProps {
  item: ReviewItem;
  isTop: boolean;           // only the top card is interactive
  onRate: (itemId: string, rating: 1 | 2 | 3 | 4) => void;
  stackIndex: number;       // 0=top, 1=second, etc. Used for visual stacking offset
}
```

The component must:

1. Maintain state: `revealed: boolean` (false by default — implements active recall pattern per REVIEW-02 decision)

2. Use framer-motion for the drag gesture:
   ```typescript
   import { motion, useMotionValue, useTransform, useAnimation } from "framer-motion";
   const x = useMotionValue(0);
   const rotate = useTransform(x, [-200, 200], [-30, 30]);
   const opacity = useTransform(x, [-200, -100, 0, 100, 200], [0, 1, 1, 1, 0]);
   const controls = useAnimation();
   ```

3. Visual stacking for non-top cards (shows a peek of cards below):
   ```typescript
   const stackOffset = stackIndex * 4; // px offset per card in stack
   const stackScale = 1 - stackIndex * 0.02; // slight scale reduction
   ```

4. Render:
   ```tsx
   <motion.div
     style={{ x, rotate, opacity, scale: stackScale, translateY: -stackOffset }}
     drag={isTop ? "x" : false}
     dragConstraints={{ left: 0, right: 0 }}
     dragElastic={0.9}
     onDragEnd={(_, info) => {
       if (!isTop) return;
       if (info.offset.x > 100) {
         // 右滑 = Easy
         controls.start({ x: 500, opacity: 0 }).then(() => onRate(item.itemId, 4));
       } else if (info.offset.x < -100) {
         // 左滑 = Again
         controls.start({ x: -500, opacity: 0 }).then(() => onRate(item.itemId, 1));
       } else {
         controls.start({ x: 0 }); // 弹回中心
       }
     }}
     animate={controls}
     className="absolute w-full max-w-md bg-white rounded-2xl shadow-lg border border-gray-200 p-6 cursor-grab active:cursor-grabbing select-none"
   >
   ```

5. Card content:
   - Header: domain badge (top-left) + review count badge (top-right)
   - Title: `<h2 className="text-xl font-bold text-center mt-4 mb-6">{item.title}</h2>`
   - If NOT revealed: show "点击揭示内容" button centered
   - If revealed: show full content text (preserve newlines with `whitespace-pre-wrap`), source link if present, then `<RatingButtons onRate={(r) => onRate(item.itemId, r)} disabled={!isTop} />`

6. Clicking anywhere on the card (when not revealed) calls `setRevealed(true)`. Do NOT use `onClick` on the entire motion.div — instead, wrap the "点击揭示" section in a button. Otherwise the click will conflict with drag.

Create `src/components/review/ReviewSession.tsx` as a client component.

The component has NO props — it is self-contained. It must:

1. Fetch the access token using supabase client:
   ```typescript
   import { createClient } from "@supabase/supabase-js";
   const supabase = createClient(
     process.env.NEXT_PUBLIC_SUPABASE_URL!,
     process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!
   );
   const { data: { session } } = await supabase.auth.getSession();
   const token = session?.access_token;
   ```
   Do this inside a `useEffect` on mount.

2. Maintain state:
   - `items: ReviewItem[]` — due items (fetched from /api/review/today)
   - `currentIndex: number` — index of the top card (starts at 0)
   - `loading: boolean`
   - `error: string | null`
   - `consecutiveEasyCount: number` — for anti-cheat nudge (FSRS-03 decision)
   - `lastRatedItem: { itemId: string; rating: number; } | null` — for undo feature (FSRS-04 decision)
   - `undoTimeoutId: ReturnType<typeof setTimeout> | null`
   - `showUndoBar: boolean`
   - `showEasyNudge: boolean`

3. `fetchDueItems()` async function:
   - GET /api/review/today with Bearer token
   - Set items state
   - Clear loading

4. `handleRate(itemId: string, rating: 1 | 2 | 3 | 4)` async function:
   ```typescript
   // 步骤 1: 乐观更新 UI（先移除卡片，避免等待 API）
   setCurrentIndex(prev => prev + 1);

   // 步骤 2: Anti-cheat nudge (FSRS-03 decision)
   if (rating === 4) {
     const newCount = consecutiveEasyCount + 1;
     setConsecutiveEasyCount(newCount);
     if (newCount > 0 && newCount % 5 === 0) {
       setShowEasyNudge(true);
       setTimeout(() => setShowEasyNudge(false), 4000);
     }
   } else {
     setConsecutiveEasyCount(0);
   }

   // 步骤 3: 设置撤销窗口 (FSRS-04 decision, 1 minute)
   setLastRatedItem({ itemId, rating });
   setShowUndoBar(true);
   const tid = setTimeout(() => {
     setShowUndoBar(false);
     setLastRatedItem(null);
   }, 60000); // 1 minute
   setUndoTimeoutId(tid);

   // 步骤 4: POST to API (do not await before moving card)
   try {
     await fetch("/api/review/rate", {
       method: "POST",
       headers: { "Content-Type": "application/json", "Authorization": `Bearer ${token}` },
       body: JSON.stringify({ knowledgeItemId: itemId, rating }),
     });
   } catch (err) {
     console.error("评分提交失败:", err);
   }
   ```

5. `handleUndo(correctedRating: 1 | 2 | 3 | 4)` function:
   - If `lastRatedItem` is set, POST /api/review/rate with corrected rating
   - Clear undo state
   - Clear timeout

6. Render:
   - If loading: "加载中..."
   - If error: error message
   - If all items reviewed (currentIndex >= items.length):
     ```tsx
     <div className="text-center py-20">
       <div className="text-4xl mb-4">🎉</div>
       <h2 className="text-2xl font-bold mb-2">今日复习完成！</h2>
       <p className="text-gray-600 mb-6">已完成 {items.length} 个知识条目的复习</p>
       <Link href="/library" className="...">返回知识库</Link>
     </div>
     ```
   - Otherwise, render card stack:
     ```tsx
     <div className="relative flex flex-col items-center justify-center min-h-[60vh]">
       {/* Undo bar at top */}
       {showUndoBar && lastRatedItem && (
         <div className="fixed top-4 right-4 bg-gray-800 text-white rounded-lg p-3 flex items-center gap-3 z-50">
           <span className="text-sm">刚才评错了？</span>
           <RatingButtons onRate={handleUndo} disabled={false} />
         </div>
       )}

       {/* Easy nudge */}
       {showEasyNudge && (
         <div className="fixed top-16 right-4 bg-yellow-100 border border-yellow-300 rounded-lg p-3 text-sm text-yellow-800 z-50 max-w-xs">
           真的这么轻松吗？如果下次忘了，进度会重置哦。
         </div>
       )}

       {/* Progress indicator */}
       <p className="text-gray-500 text-sm mb-4">
         {currentIndex} / {items.length} 已完成
       </p>

       {/* Card stack — render top 3 cards */}
       <div className="relative w-full max-w-md h-[400px]">
         {items.slice(currentIndex, currentIndex + 3).reverse().map((item, i) => (
           <ReviewCard
             key={item.itemId}
             item={item}
             isTop={i === (Math.min(3, items.length - currentIndex) - 1)}
             onRate={handleRate}
             stackIndex={Math.min(3, items.length - currentIndex) - 1 - i}
           />
         ))}
       </div>
     </div>
     ```
