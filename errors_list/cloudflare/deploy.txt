# Cloudflare 部署错误记录

## 2026-03-25 错误

**错误信息：**
```
ERROR Could not find compiled Open Next config, did you run the build command?
```

**根本原因：**
wrangler.jsonc 中的 build.command 设置为 "npm run build"，这只会运行 "next build"。
但部署到 Cloudflare Workers 需要运行 "opennextjs-cloudflare build" 来生成 .open-next/worker.js。

**解决方案：**
修改 wrangler.jsonc 中的 build.command 为 "npm run build:cloudflare"

**状态：** 已修复

## 2026-03-26 部署尝试 #1

**错误信息：**
```
./next.config.ts:44:3
Type error: Object literal may only specify known properties, but 'hideSourceMaps' does not exist in type 'SentryBuildOptions'. Did you mean to write 'sourcemaps'?
```

**根本原因：**
`@sentry/nextjs` 包的配置选项已更改。`hideSourceMaps` 和 `disableLogger` 选项已被弃用。

**解决方案：**
更新 Sentry 配置使用新的 `sourcemaps` 和 `bundleSizeOptimizations` 选项。

**状态：** 已修复

## 2026-03-26 部署尝试 #2

**错误信息：**
```
./next.config.ts:50:5
Type error: Object literal may only specify known properties, and 'excludeReplayCanvas' does not exist in type '{ excludeDebugStatements?: boolean | undefined; excludeTracing?: boolean | undefined; excludeReplayShadowDom?: boolean | undefined; excludeReplayIframe?: boolean | undefined; excludeReplayWorker?: boolean | undefined; }'.
```

**根本原因：**
`bundleSizeOptimizations.excludeReplayCanvas` 不是有效的选项。根据 TypeScript 类型定义，
有效的选项是：excludeDebugStatements, excludeTracing, excludeReplayShadowDom, excludeReplayIframe, excludeReplayWorker。

**解决方案：**
移除 `excludeReplayCanvas: true` 配置项。

**状态：** 已修复，等待重新部署
